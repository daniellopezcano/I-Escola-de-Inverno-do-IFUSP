# Exemplos Práticos em JAX

Este diretório contém os notebooks e códigos fonte para as sessões práticas da Escola de Inverno.

## Como usar no Google Colab
Cada notebook na pasta \`notebooks/\` possui um link "Open in Colab" no topo. Basta clicar para rodar o código sem instalação local.

## Estrutura
- \`notebooks/\`: Notebooks Jupyter prontos para o Colab.
- \`src/\`: Módulos JAX reutilizáveis (ex: funções de perda, emuladores simples) para manter os notebooks limpos.
